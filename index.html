วิชาเว็บไซต์เพื่อธุรกิจ
เรื่อง การสร้าง Web Page ด้วยภาษา html จำนวน 10 ข้อ
โดย อ.ธนะวัฒน์ เลิศประเสริฐ วิทยาลัยชุมชนนราธิวาส
คำสั่ง เลือก หรือ เติมคำตอบที่ถูกต้องที่สุด
ข้อที่ 1)
การนำข้อมูลจากเครื่องคอมพิวเตอร์เราไปไว้บนอินเตอร์เน็ตเราเรียกกว่าอะไร
   Download
   Upload
   Reload
   Move
ข้อที่ 2)
ข้อใดเป็น Tag เกี่ยวกับลักษณะตัวอักษรตัวหนา
   <B>........</B>
   <I>........</I>
   <U>........</U>
   <BLINK>........</BLINK>
ข้อที่ 3)
ข้อใดเป็น Tag คำสั่งแสดงภาพบน Homepage
   <IMAGE SRC=URL หรือ ชื่อไฟล์ภาพ>
   <IMG SRCE=URL หรือ ชื่อไฟล์ภาพ>
   <IMG SRC=URL หรือ ชื่อไฟล์ภาพ>
   <IMGSRC=URL หรือ ชื่อไฟล์ภาพ>
ข้อที่ 4)
ข้อใดเป็น Tag คำสั่งการสร้าง Column ของ TABLE
   <TH> </TH>
   <TR> </TR>
   <TD> </TD>
   <TABLE> </TABLE>
ข้อที่ 5)
ข้อใดเป็นการเขียนคำสั่ง Links ให้เปิดเว็บโรงเรียนซางตาครู้สศึกษา
   <a href=http://www.sc.ac.th>ซางตาครู้สศึกษา</a>
   <a href=http://www.sc.co.th>ซางตาครู้สศึกษา</a>
   <a href=http://www.sc.ac.th>ซางตาครู้สศึกษา</b>
   <a href=http://www.cs.ac.th>ซางตาครู้สศึกษา</a>
ข้อที่ 6)
คำสั่งขึ้นบันทัดใหม่คือข้อใด
   <BR>
   <HR>
   <TR>
   <B>
ข้อที่ 7)
ข้อใดเป็นการใส่สีพื้นหลังให้กับ Homepage
   <bodybgcolor=yellow>
   <body bgcolor=yellow>
   <body bgcoler=yellow>
   <body background color=yellow>
ข้อที่ 8)
ข้อใดเป็นการใส่รูปภาพเป็นพื้นหลัง
   <body bgcolor=yellow>
   <body background=yellow>
   <body bgcoler=img1.gif>
   <body background=img1.jpg>
ข้อที่ 9)
คำสั่งตีเส้นคือข้อใด
   <BR>
